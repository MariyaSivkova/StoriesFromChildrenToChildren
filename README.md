# StoriesFromChildrenToChildren
SoftUni Project

## Description
**Stories from Children to Children** is a web-based platform where children can share and enjoy audiobooks recorded by other children. The goal of the project is to encourage creativity, improve reading skills, and foster a sense of community among young users.

The platform allows users to:
- TODO: Upload audiobooks recorded by children (upload is still work in progress)
- Browse and download available stories
- Listen to audio content directly through the website
- Use a simple and child-friendly interface

---

## Technologies Used

- C#
- ASP.NET Core 8
- ASP.NET MVC (Model-View-Controller)
- Entity Framework Core
- SQL Server
- HTML, CSS, JavaScript

---

## Setup Instructions and Usage

### Prerequisites
Make sure you have the following installed:

- .NET 8 SDK
- SQL Server
- Visual Studio 

---

### Installation Steps

1. Clone the repository:
   git clone: https://github.com/MariyaSivkova/StoriesFromChildrenToChildren.git
2. Open Visual Studio Solution.
3. Open Package Manager Console and run Update-Database to create database for the application.
4. Build and run the application.
